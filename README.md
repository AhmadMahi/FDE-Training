# FDE GenAI Playbook

A comprehensive field reference guide for Forward Deployed Engineers covering Generative AI concepts, tools, and production best practices.

## What's Covered

**Foundations**
- Transformers, self-attention, MoE architectures, and multimodal LLMs
- Prompt engineering techniques: CoT, ReAct, Tree of Thought, Self-Consistency, and more

**RAG & Context Engineering**
- Context engineering vs prompt engineering
- RAG architecture, document ingestion, chunking strategies
- Embeddings, vector databases, semantic search, hybrid search, and re-ranking

**Agentic AI**
- Agent architectures: ReAct, Plan-and-Execute, Reflexion
- Single-agent vs multi-agent systems with LangGraph
- Agentic RAG, CRAG, OpenAI Agents SDK
- Agent protocols: A2A (Agent-to-Agent) and MCP (Model Context Protocol)

**LLM Orchestration**
- LangChain architecture, LCEL, memory types, tool calling
- LangGraph stateful workflows and graph-based execution

**Security & Guardrails**
- OWASP LLM Top 10, prompt injection, input validation
- Guardrail design, output filtering, data leakage prevention, responsible AI

**Evaluation**
- RAGAS metrics: faithfulness, relevance, context precision/recall
- Golden datasets, hallucination detection, LLM-as-judge
- Automated eval tools: Ragas, DeepEval, TruLens, PromptFoo

**LLMOps**
- MLOps vs LLMOps, fine-tuning methods (LoRA, QLoRA, DPO)
- Deployment architectures, prompt versioning, cost optimization
- Observability, monitoring, drift detection, continuous improvement loops

## Who This Is For

Forward Deployed Engineers who need a quick, reliable reference while building and deploying GenAI systems in production.

## How to Use

Open the `.docx` file in Microsoft Word or Google Docs. Each section is self-contained — jump directly to the topic you need using the Table of Contents on page 2.
